# Vorssaint Homebrew Tap

Install [Vorssaint](https://github.com/vorssaint/vorssaint-utils) with Homebrew:

```sh
brew install --cask vorssaint/tap/vorssaint
```

That command taps this repository and installs the latest signed, notarized
release. To update later:

```sh
brew upgrade --cask vorssaint
```

Vorssaint is the free, open-source menu bar toolkit for macOS. License:
PolyForm Noncommercial.
